# Branch(Plugin) 1 vimVundlePlugin

一个vim编辑器的插件管理工具。插件安装如下：

## 方法一
1：配置.vimrc文件

将.vimrc文件拷贝到环境目录~/下面

2：安装vundle插件

将bundle/vundle下面的所有文件拷贝到~/.vim/bundle/vundle下面

3：安装常用的插件

命令如下
```
vim

:BundleInstall
```

## 方法二

因为考虑到有些环境的网络场景而不能顺利完成常用插件的安装，所以这里已经下载好了常用的插件。所有配置好.vimrc后，将整个bundle目录拷贝到环境的~/.vim/下面，就可以使用炫酷的vim编辑器了。
